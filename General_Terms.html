  • รับผิดชอบการใช้งานบัญชี ผู้ใช้ และ API Key ของตน

เวอร์ชันย่อ (เน้น "การศึกษาและพัฒนา")

ขอบเขตและวัตถุประสงค์

เงื่อนไขฉบับนี้กำหนดกรอบการใช้บริการเชื่อมต่อ IoT/M2M ของ GtgAlpha เพื่อใช้ในการศึกษา ทดลอง และพัฒนาระบบติดตาม/IoT เท่านั้น ไม่เน้นการใช้งานเชิงพาณิชย์เต็มรูปแบบ บริการครอบคลุมการเชื่อมต่อเครือข่ายมือถือสำหรับอุปกรณ์ IoT แพลตฟอร์มบริหารจัดการซิม และ API/เครื่องมือสำหรับนักพัฒนาที่ให้บริการผ่านเว็บไซต์ https://gtsalpha.in.th.

เนื้อหาบริการและอายุซิม

ซิมของ GtgAlpha ให้สิทธิ์ทราฟฟิกข้อมูลและ SMS จำนวนจำกัดต่อซิม เพื่อใช้ทดสอบอุปกรณ์และแพลตฟอร์ม เช่น ดาต้าเริ่มต้นและจำนวน SMS ตามที่ระบุในแพ็กเกจทดลอง ระยะเวลาใช้งานของแต่ละซิมถูกกำหนดชัดเจน (เช่น 2 ปี) เพื่อเพียงพอสำหรับการวิจัย พัฒนา และทดสอบภาคสนาม โดยอาจมีช่วงเวลา Grace Period เพื่อเตรียมระบบก่อนเริ่มนับอายุจริง.

การใช้งานที่อนุญาต

ลูกค้าสามารถใช้ซิมและบริการเพื่อ: ทดสอบฟังก์ชัน GPS/การติดตามรถ ทดลองโปรโตคอล MQTT/HTTP และพัฒนาเฟิร์มแวร์หรือแพลตฟอร์มบริหารอุปกรณ์ ห้ามนำบริการไปรีเซลเป็นบริการโทรคมนาคมเชิงพาณิชย์ให้บุคคลที่สาม หรือใช้งานผิดกฎหมาย/ผิดระเบียบ.

ซอฟต์แวร์ แพลตฟอร์ม และสิทธิใช้งาน

GtgAlpha ให้สิทธิใช้งานซอฟต์แวร์ แดชบอร์ด และ API แบบไม่ผูกขาด โอนไม่ได้ และใช้ได้เฉพาะตามวัตถุประสงค์การศึกษา/พัฒนาที่ระบุในเอกสารนี้ หากมีการอัปเดต แพตช์ หรือเปลี่ยนเวอร์ชัน นักพัฒนาต้องรับรองว่าอุปกรณ์ของตนสามารถรองรับการเปลี่ยนแปลง และเราอาจปรับบริการเพื่อความปลอดภัยและเสถียรภาพ.

ความรับผิด ข้อมูล และกฎหมาย

บริการให้ในลักษณะ “ตามที่มีและตามที่ให้” เพื่อการทดลอง อาจมีการหยุดชะงักหรือข้อจำกัดด้านคุณภาพโดยไม่ถือเป็นการผิดสัญญาเชิงพาณิชย์เต็มรูปแบบ ทั้ง GtgAlpha และลูกค้าต้องปฏิบัติตามกฎหมายที่เกี่ยวข้องกับโทรคมนาคมและข้อมูลส่วนบุคคล และเว็บไซต์อย่างเป็นทางการของเรา คือ https://gtsalpha.in.th สำหรับข้อมูลอัปเดตเพิ่มเติม.
